## 3. Project Overview

The **Moringa GenAI Go Toolkit** is a beginner-level Go project developed to demonstrate how **Generative AI can assist students in learning a new programming language**.

The project combines:

* A simple Go application
* AI-assisted learning and troubleshooting
* Proper technical documentation

The main goal is **not complexity**, but to show **how AI supports understanding, problem-solving, and documentation in software development**.

## 6. Code Explanation (main.go)

### Source Code

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

### Explanation

* `package main` defines the program as executable.
* `import "fmt"` allows formatted output.
* `func main()` is the entry point of the program.
* `fmt.Println()` prints output to the terminal.

This simple program was used to **validate Go installation** and understand execution flow.

## 8. How to Run the Project

### Requirements

* Go installed
* Git installed
* Terminal access

### Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/6manusigei3/moringa-genai-go-toolkit.git
   ```

2. Navigate into the project folder:

   ```bash
   cd moringa-genai-go-toolkit
   ```

3. Run the program:

   ```bash
   go run main.go
   ```

4. Output:

   ```
   Hello, World!
   ```
